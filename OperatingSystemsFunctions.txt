There are numerous functions of an operating system.
One of an operating systems function is to provide a user interface.  Most of us are familiar with a graphical user interface.  It allows us to use a mouse and to see images on screen.  A command line interface is typically for more advanced users.  
Another function of an operating system is to start and shut down a computer.  After pressing the power button on your device, it sends electricity to your CPU which runs the firmware instructions.  After which it will perform a series of tests to confirm that everything in your device is functional.  If everything is functional then the kernel of the operating system will load after which the log in will appear.
